Next-Shop

Full-stack ecommerce demo built with Spring Boot and Next.js.

Tech stack
- Spring Boot 3
- PostgreSQL
- JPA / Hibernate
- Next.js
- Zustand
- Tailwind

Features
- Product catalog
- Shopping cart
- Quantity merge
- Checkout page
- REST API architecture

./mvnw clean compile
mvn spring-boot:run
